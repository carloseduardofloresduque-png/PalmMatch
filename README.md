# Palm Match Hi5

Scratch memory card matching game + high five timing challenge!

- Flip cards to find matches  
- On match: Hands race in — press SPACE at perfect time for bonus points!  
- Perfect = +20 pts, Good = +10, Miss = 0  

## How to Play
1. Download the .sb3 file  
2. Open Scratch (scratch.mit.edu) → File → Load from your computer  
3. Play!
